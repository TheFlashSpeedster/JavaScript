Object in JavaScript

In JavaScript, an object is a collection of properties, and a property is an association between a name (or key) and a value. Objects are used to store multiple values in a single variable.

Syntax

let objectName = {
    property1: value1,
    property2: value2,
    property3: value3,
  };

  // Accessing Properties
  objectName.propertyName; // whole object
  objectName.property1; // property1
  objectName.property2; // property2
  objectName.property3; // property3
