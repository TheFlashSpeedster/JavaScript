Common Array Methods

  1. push(item) - add item at end
  2. pop() - remove item at end
  3. shift(item) - add item at front
  4. unshift() - remove item at front
  5. splice(startIndex, deleteCount, newitem1, newitem2, ...) - add or remove elements
  6. toString() - Convert into string (array without brackets)
  7. sort() - sort elements
  8. valueOf() - make a copy of array
