Concatenation

'Hello' + 'World' : 'HelloWorld'

'Hello' + 5 + 'World' : 'Hello5World'

'Hello' + 5 + 4 + 'World' : 'Hello54World'

'Hello' + (5 + 4) + 'World' : 'Hello9World'

5 + 4 : 9

  • string + string = string
  • number + number = number
